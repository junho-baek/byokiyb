# Bring Your Own Key In Your Bed (BYOKIYB)

Local-first credential handoff for agent-operated projects.

- CLI name: `byokiyb`
- Working category: BYOK gateway / safe credential handoff
- V1: CLI + mobile one-time intake + `.env.local` writer
- Agents receive only presence/status/metadata, never raw values.

Seed/spec: [`SEED.yaml`](./SEED.yaml)
